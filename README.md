# ReadME

Commodity Correlation Dashboard

Project Description
This Streamlit application analyzes the statistical relationship between various commodities, market indices, and equities. By calculating the Pearson correlation coefficient and providing normalized price charts, the tool helps users identify how assets move in relation to one another over specific timeframes.

Target Users
Finance students and academic researchers.

Commodity traders looking for hedging opportunities.

Portfolio managers interested in asset diversification.

Final Features
Asset Comparison: Select from different futures, ETFs, and mining stocks.

Lookback Selection: Toggle between multiple timeframes (6mo to 5y) using the sidebar.

Correlation Analysis: Real-time calculation of the correlation score between two chosen assets.

Normalized Performance Chart: Visual price comparison starting from a base of 100 to show relative growth/decline.

Educational Expander: Built-in guide to help users interpret correlation scores.

Deployed App Link:
https://correlationdashboard.streamlit.app/

How to Use
Use the dropdown menus to pick two different assets.

Select your desired lookback period from the sidebar radio buttons.

Review the Correlation Score to see the statistical strength of the relationship.

Compare the Price Comparison chart to see which asset outperformed the other over the period.
